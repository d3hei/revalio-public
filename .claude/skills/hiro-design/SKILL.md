Read the `readme.md` file within this skill, and explore the other available files (`styles.css` + `tokens/`, the `guidelines/*.card.html` specimen cards, `components/<group>/` primitives, and `ui_kits/<product>/`).

If creating visual artifacts (slides, mocks, throwaway prototypes, etc), copy assets out and create static HTML files for the user to view. If working on production code, copy assets and read the rules here to become an expert in designing with the Hiro brand.

Key brand cues to honor: a clean white / warm-paper canvas, near-black ink, **violet `#5546FF`** primary + **Bitcoin orange `#F7931A`** accents; a geometric-grotesque + monospace type pairing; **UPPERCASE display headlines ending in a period**; uppercase **mono eyebrows** (`/ Tools`), **bracketed controls** (`[ Subscribe ]`), and the signature **terminal / `$ tree` file-tree** motifs (`├──`, `└──`, `# comments`). Boxy corners, hairline borders, restrained shadows, quick mechanical motion, **no emoji, no gradients**.

If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.
